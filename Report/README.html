How to Use

  • prepare environment

    • cd Back-end

      • npm install --save-dev @openzeppelin/contracts
      • npm install dotenv

    • cd Front-end

      • npm install

  • run tests

    • cd Back-end

      • npx hardhat test

  • deploy contract

    • cd Back-end

      • npx hardhat run scripts/deploy.ts --network etherdata

  • start webpage

    • cd Front-end

      • npm start
